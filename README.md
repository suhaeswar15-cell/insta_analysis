# Instagram Engagement Analysis for Alfido Tech

This project delivers a professional Instagram engagement analysis using a Kaggle-style dataset. It includes data loading, cleaning, feature engineering, time-based and content-based insights, follower trend evaluation, and a recommended weekly content calendar.

## Contents
- `instagram_engagement_analysis.ipynb`: analysis notebook
- `requirements.txt`: Python dependencies
- `strategy_document.md`: one-page engagement strategy for Alfido Tech

## How to use
1. Place the Instagram dataset file in the project folder.
2. Run `instagram_engagement_analysis.ipynb` in Jupyter Notebook or JupyterLab.
3. Review the charts and strategy recommendations generated from the data.

## Key focus areas
- Best posting hours and days
- Content type performance
- Hashtag effectiveness
- Follower and engagement trends
- Actionable weekly content calendar
